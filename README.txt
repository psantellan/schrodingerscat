Comentarios sobre la página web:

1) No todos los productos se pueden agregar al carrito de compras, solo los productos ingresados mediante el administrador.
2) No pude crear botones para editar los items ingresados, tampoco para borrarlos. Solo se puede ingresar productos.
3) Tampoco pude crear botones en el carrito de compras para eliminar los productos y para finalizar la compra. Como en el punto anterior, solo se puede agregar.
